# DN.COM Portfolio - Project Summary

## Project Overview

DN.COM is a personal portfolio website built for Demetre Nutsubidze, a frontend developer based in Tbilisi, Georgia. The website showcases his skills, projects, and experience in a modern, minimalist design with smooth animations.

## Design Features

### Visual Style
- **Color Scheme**: Black background with white text and yellow/gold accents (#D4AF37)
- **Typography**: Modern sans-serif fonts (Inter, SF Pro Display, system fonts)
- **Layout**: Dark theme with high contrast and smooth transitions

### Animations
- **Framer Motion**: Smooth page transitions and scroll animations
- **Custom Cursor**: Animated cursor with magnetic effects
- **Card Animations**: Hover effects on project cards and skills
- **Scroll Effects**: Reveal animations for sections as you scroll

## Technologies Used

- **React 19** - UI library
- **Vite 7** - Build tool
- **Framer Motion** - Animations
- **Tailwind CSS** - Styling
- **Lucide React** - Icons
- **gh-pages** - Deployment

## Responsive Design

The website is fully responsive for:
- Desktop (1200px+)
- Tablet (768px - 1199px)
- Mobile (320px - 767px)

## Performance Optimization

- All images are optimized for web
- Code is minified and compressed
- CSS and JavaScript are bundled efficiently
- Images have appropriate lazy loading

## Installation

```bash
# Clone the repository
git clone https://github.com/Nutsubidze423/Dn.Com.git

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
